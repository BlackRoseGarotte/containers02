## Цель работы

Задача состоит в создании виртуальной машины на базе Debian без графического интерфейса с помощью гипервизора QEMU и диска формата qcow2. Внутри системы нужно установить и настроить стек LAMP, включающий веб-сервер Apache, интерпретатор PHP и СУБД MariaDB, а также создать конфигурации для двух сайтов. Завершающим шагом является развертывание CMS WordPress и панели PhpMyAdmin с проверкой их работы через проброшенные порты.

После завершения установки Qemu, был скачан образ Debian и перенесен в папку `dvd`.

pic

Был создан файл `.gitignore`. Он необходим для исключения из отслеживания системой контроля версий Git тяжелых и временных файлов, таких как образы виртуальных дисков, ISO-образы и архивы, которые не должны храниться в репозитории.

pic

Далее была использована команда `qemu-img create -f qcow2 debian.qcow2 8G`. Эта команда создает пустой виртуальный жесткий диск размером 8 ГБ в динамическом формате qcow2 с именем debian.qcow2.

pic

Для установки Debian на виртуальную машину было необходимо ввести в консоль команду `qemu-system-x86_64 -hda debian.qcow2 -cdrom dvd/debian.iso -boot d -m 2G`, перед этим переименовав файл с ISO-образом. После этого появилось окно установщика ОС Debian.
